1- >> vi /etc/ssh/sshd_config (edit the port number to 2222)
2- >> vi /etc/selinux/config  (edit selinux from enforcing to disable)
3- >> setenforce 0
4- >> sestatus (if it is still enabled then >> reboot)
5- if you stoped the firewall =>  >> systemctl enable firewalld
				  >> systemctl start firewalld
6- >> firewall-cmd --add-port=2222/tcp --permamemt
7- >> firewall-cmd --remove-service=ssh --permanent
8- >> semanage port -a -t ssh_port_t -p tcp 2222 
	if semanage isn't installed then >> dnf install policycoreutils-python-utils 
	and try the command #8 again
9- >> reboot
10- from the other machine >> ssh username@ipaddress -p 2222
